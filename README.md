# Node Firebase Boilerplate.

![Node-Firebase-Boilerplate](https://myhappyagency.com/blog/wp-content/uploads/2020/01/node-Firebase.png)

Minimalist, simple boilerplate for Node & Firebase.js.

## Installation

Install dependencies:
```
npm i
```
Serve project:
```
npm run start
```
